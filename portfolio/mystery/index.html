pipeline

  • Weekly XLSX shopping lists define available produce overage
  • SSH-tunnelled MySQL queries fetch prices, customers, and box tiers
  • Pluggable strategy architecture: swap algorithms without changing plumbing
  • Tab-delimited output imports directly back into the Laravel app
